A complete **Zomato Restaurant Analysis** project using Python.
500 Indian restaurants across 15 cities with 8 visualizations and deep insights.

---

## 🎯 Project Overview

> Analyze 500 Zomato restaurants to understand **which cities have best food**,
> **which cuisines are most popular**, how **price affects ratings**,
> and what makes a restaurant **highly rated and voted**.

    cd zomato-restaurant-analysis
    pip install -r requirements.txt

## 🔍 Key Findings

| Finding | Insight |
|---------|---------|
| ⭐ Best Rated City | Surat has highest avg restaurant rating |
| 💰 Price vs Rating | Premium restaurants rated slightly better |
| 📱 Online Boost | Online order restaurants get more votes |
| 🍜 Top Cuisine | Cafe style restaurants most common |
| 🏆 Fine Dining | Highest avg cost — ₹2000+ per visit |
| 📈 Growth | Restaurant openings peaked in 2018-2020 |
| 🗳️ Votes Matter | Higher votes = more credible ratings |
| 🍛 Famous Dish | Biryani most common famous dish |
